How it works

Five-stage automated pipeline

01

Media Library Access

The agent connects to a designated Google Drive folder and retrieves media files — images and videos — ready for publishing. Files are sourced, downloaded, and converted for each platform's requirements automatically. No manual file handling required.

02

AI Content Generation

Claude Vision analyzes each media file and generates platform-optimized captions, hashtags, comments, and reply templates. Content is tailored per platform — cinematic long-form for Pinterest, punchy hooks for Instagram Reels, concise text for X — using a structured prompt system refined over time.

03

Scheduled Publishing

Posts are published automatically via official platform APIs at scheduled times. The schedule is not fixed — it is driven by performance data, adjusted dynamically to favour the windows that historically deliver the highest engagement for each platform and content type.

04

Performance Analytics Collection

After publishing, the agent polls each platform's analytics API to collect impressions, reach, likes, saves, comments, and shares for every post. All metrics are logged to a structured Google Sheets analytics hub and stored for trend analysis.

05

Adaptive Strategy Optimization

Performance data feeds back into the pipeline. The agent identifies top-performing content formats, posting times, caption styles, and hashtag sets. Future scheduling and content generation are adjusted accordingly — creating a continuously improving loop without manual intervention.

Capabilities

Built for autonomous operation

Every component of the pipeline is designed to run without manual input, from media retrieval through to strategy refinement.

☁️

Google Drive Media Management

Connects directly to a designated Google Drive folder. Retrieves, converts, and routes media files to the correct publishing pipeline per platform — images to Pinterest and Instagram, videos to Reels and TikTok.

🤖

AI Caption & Content Creation

Uses Claude Vision to analyze images and generate captions, hashtags, alt text, and comment replies. Each output is formatted to match platform-specific style guides and adjusted based on what has performed well historically.

📊

Post Performance Tracking

Retrieves engagement metrics from each platform's official API after every post. Impressions, reach, saves, likes, and shares are logged automatically to a structured analytics sheet for every piece of content published.

⏱️

Adaptive Scheduling

Analyses historical performance data to identify the posting windows that drive the most engagement on each platform. The publishing schedule updates dynamically — no fixed cron, no manual time-slot guessing.

🔁

Continuous Learning Loop

Each publish-and-measure cycle feeds data back into the content and scheduling decisions for the next cycle. Over time the agent learns which formats, tones, and times work best per platform and refines its approach automatically.

🔐

Secure OAuth 2.0 Access

All platform connections use official OAuth 2.0 authorization flows. No passwords are stored. Access tokens are scoped to publishing and analytics only, and are managed exclusively within the private n8n Cloud instance.
